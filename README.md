# 💰 Personal Finance Dashboard

## Overview
This project analyses 12 months of personal income, expenses and savings using Python.
Built to demonstrate financial data analysis skills.

## Charts Included
-  Monthly Income vs Expenses
-  Net Savings Trend
-  Annual Expense Breakdown
-  Monthly Savings Rate

## Tools Used
- Python
- Pandas
- Matplotlib
- NumPy
- Jupyter Notebook

## Key Findings
- Total Annual Income: £31,200
- Total Annual Expenses: £27,135
- Total Net Savings: £4,065
- Average Savings Rate: 13%
- Rent is the largest expense at 35% of spending
